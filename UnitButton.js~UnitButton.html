UnitButton

react~React.Component → UnitButton

UnitButton is a hover button-live element displaying its label marked up with a div of "button" class and "selected" if active.

React Props:

Name Type Attribute Description
label string can be any string to display in the div
value string the label, but needed if the value differs from the label
selected boolean true if the container div knows this button is active
highlight function a function, something for the button to do onClick
