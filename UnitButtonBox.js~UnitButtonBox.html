UnitButtonBox

Extends:

react~React.Component → UnitButtonBox

UnitButtonBox has left/right mouse button controls, as well as clickable, hoverable, buttons that are highlighted when active.

React Props:

Name Type Attribute Description
buttonValues string[] strings to be mapped to UnitButtons
title string the unit title for this set of values
classname string a string to assign the class name for the button box
onEvent function a handleUnitChange function from Main to deal with unit change.
change function flips the selected image index forward or back in DataCanvasContainer (parent) so the selected image updates in the DataCanvas
